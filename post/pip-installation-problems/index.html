pip安装路上的各种坑

2018-06-14 Python

最近使用pip安装python扩展包的时候，又遇到了许多问题，现在一一记录下来，以备日后查询。

1. 问题由来

今天在使用pythonjupyter notebook的时候出现了问题，浏览器显示无法连接。一气之下删除了python，安装了新的python版本和pip版本。然而，现在发现遭遇了更大的问题。

自从重装之后，安装许多扩展包均会出现问题，如numpy、pandas、scrapy、mysqlclient、jupyter notebook，出现的问题都是一样：

error: Microsoft Visual C++ 14.0 is required. Get it with "Microsoft Visual
C++ Build Tools": http://landinghub.visualstudio.com/visual-cpp-build-tools

既然error中已经给了解决方法，那么首先还是要根据这一天指示来。注意，第一个坑出现了，如果直接进入这个这个链接所在的地址，那么你会大失所望。等待了很久，结果显示Page not found。

既然此路不同，那么只能选择其他方法了。

2. 坑之一——Visual Studio 2015

搜了许多网上教程，其中有说需要安装visualcppbuildtools_full.exe，折腾了许久，按照默认参数一步一步安装了这个软件。然而我还是太天真了，看着安装的各种陌生的软件，菜鸟如我一下就懵逼了。先不说这些软件差不多占了3.3G空间，还在快捷栏中添加了许多完全不认识的软件和文件。更令人伤心的是，折腾了这么久，还是无法使用pip install xxx，依旧显示同样的问题。完全不能忍，一气之下删除了所有刚才安装了Visual Studio各种软件。
