Windows下生成git ssh公钥

2018-06-13 Git

Git是一款开源版本控制系统（Version Conrol System, VCS），由大神Linus用C写成。

ssh：ssh是Secure Shell（安全外壳协议）的缩写，建立在应用层和传输层基础上的安全协议。为了便于访问github，要生成ssh公钥，这样就不用每一次访问github都要输入用户名和密码。

与Linux系统相比，Windows下生成git ssh 相对复杂，在注册github账户之后，安装git之后，可采用以下步骤生成与账户对应的ssh公钥。

1. Git bash

成功安装git后，打开git bash。

2. 生成公钥

在git bash中输入ssh-keygen -t rsa -C "example@example.com"。

采用上述命令生成 public/private rsa key pair.

接下里包括公钥存入的文件和密码等步骤，这几步均可采用默认即可。

Enter file in which to save the key (/c/Users/Administrator/.ssh/id_rsa):
Enter passphrase (empty for no passphrase):
Enter same passphrase again:

3. 将key添加到github账户中

打开id_rsa.pub,复制其中内容；

找到github账户settings中添加New SSH key,将复制的内容添加到Key区域。

4. 测试

输入ssh -T git@gitub.com进行测试，显示一下内容时表明已经成功将公钥添加到github账户。

Hi xxx! You've successfully authenticated, but GitHub does not provide shell access.
