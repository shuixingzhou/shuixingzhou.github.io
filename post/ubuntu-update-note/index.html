Ubuntu 18.04LTS更新日记

2018-05-05

两年前因为好奇装上了Ubuntu16.04LTS，之后一发不可收拾，从此走上了不折腾不舒服斯基的道路。参考网上资料、购买书籍或者自己瞎几把捣鼓，加固态硬盘，扩充内存条，装双系统等，在自己条件可及的情况下把这台老当益壮的惠普折腾的够呛，而自己也在Linux这个坑里越陷越深。 4月26号发布了期待已久的的Ubuntu18.04LTS，第一时间也上手尝试了一下。

1. 关于Ubuntu18.04LTS

Ubuntu18.04LTS是Ubuntu最新的长期支持版本，对桌面版和服务器版有长达5年的支持。Ubuntu18.04被Canonical的创始人Mark Shuttleworth命名为Bionic Beaver（仿生海狸）。

2. Ubuntu18.04LTS安装

由于之前安装的是Ubuntu16.04，而从Ubuntu16.04直接升级到Ubuntu18.04的版本7月份才发布。因此只能采用重新安装的方法。本次我采用制作U盘启动盘的方法直接安装Ubuntu。

（1）ISO映像文件下载

由于本次Ubuntu18.04只提供amdx64架构，因此直接在Ubuntu官网下载了这个版本。

（2）启动盘制作

采用官网推荐的启动盘制作程序rufus。rufus操作简单，使用方便。

（3）直接安装

插入制作成功的启动U盘后，首先进入BIOS系统设置U盘启动，然后重新启动，自动进入系统安装界面。如果对分区没有要求，直接选择下一步即可。同时，在本次发布的版本中新加了Minimal installation，相较于Normal installation，Minimal installation仅包括核心功能，即Ubuntu桌面环境以及浏览器和必备标准组件，如 GNOME apps like Terminal, System Monitor, Calculator, Gedit, Evince, and Videos (Totem)。

本次采用Normal installation。同时对分区做了调整。分区如下：

/: 30G
/boot: 600M
/swap: 2048M
/home: 剩余所有的空间

3. 新体验

相比于之前的16.04，本版本更加清爽。最大的区别之一是，取消了DASH框。桌面环境采用的是GNOME desktop 3.28，内核采用的是linux kernel 4.15。相比之前版本，使用起来并不会造成什么障碍。

4. 必备软件安装

初略体验了一把之后，要想让该系统能够正常工作，还需要添加必备的软件。

中文输入法
简单操作采用英文可以，然而日常办公还是需要中文输入法的。在Ubuntu16.04，采用搜狗输入法，上手与Windows环境没差。在本次版本中目前还不支持搜狗输入法，但是完全可以采用其他的方法。 设置的方法是：Settings —> Region & Language —> Input Source: Chinese (Intelligent Pinyin)。

Vim文本编辑器
很奇怪本次发布的新版本中竟然没有默认安装Vim文本编辑器。Vim文本编辑器是功能最强大、使用最普遍的编辑器之一，因此在发现没有自带后毫不犹豫的安装了Vim编辑器。可以直接采用sudo apt install vim进行安装。

R语言和Rstudio
作为必备了统计分析软件R语言，在之前的版本中是默认安装的。但是在新的版本需要自己手动安装。在本次安装中R和RStudio均是下载deb包手动安装。

Pycharm
新的版本中默认安装了Python2.0和Python3.0，都是最新版本。Python的IDE，相比还是Pycharm功能最为全面，使用操作也更加人性化。只是Pycharm安装需要下载deb包进行安装。

WPS
Ubuntu中自带的办公软件是LibreOffice系列，相比该办公套件WPS系列办公软件更加适合从Windows迁移过来的用户。
